## 🏗️ 项目一：四路智能抢答器 (数电)

**项目目标**：设计并制作一个供四人竞技使用的抢答器，能够准确、公正地锁定并显示优先抢答者。

### 核心技术点
* **逻辑控制**：采用 **74S373 锁存器** 实现结果的自动锁存，防止多人同时抢答导致的冲突。
* **反馈与提示**：利用 **74LS20 与非门** 构成的逻辑电路控制**蜂鸣器**和 **LED 指示灯**，在抢答成功时提供实时的声光提示。
* **自动复位**：设计了主持人一键复位逻辑，确保竞赛流程的连贯性。

### 实验设备
* 74S373, 74LS20, 发光二极管, 有源蜂鸣器, 三极管, 电阻电容等。

## 🔋 项目二：带保护的可调电流锂电池电子负载 (模电)

**项目目标**：制作一个具备稳流功能和过放保护功能的电子负载，用于锂离子电池的性能测试。

### 核心技术点
* **稳流控制 (Constant Current)**：通过 **运算放大器 (ST358)** 实时比较电流检测电阻（水泥电阻并联）上的压降与基准电压，利用负反馈调节三极管（2SD1623, ZTX1049A）的导通程度。
* **过放保护**：集成继电器控制逻辑，当电池电压低于安全阈值时自动切断负载，防止电池受损。
* **高精度反馈**：利用“虚短”和“虚断”原理设计直流放大电路，确保电流调节的线性度与稳定性。

### 实验设备
* LM358/ST358 运放, 2SD1049, 2SC1623 三极管, 继电器, 水泥电阻 (0.5Ω 并联), 电位器等。
